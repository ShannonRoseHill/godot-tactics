# godot-tactics

The purpose of this project is to learn the basics of tactical RPG development.
I'll do this by analyzing design decisions from the Godot Tactics series on the The Liquid Fire game programming blog.
This process will be a line-by-line code review with extensive commenting and documentation.

There are seventeen blog entries in the series, with the last being published on 29 August 2025.
